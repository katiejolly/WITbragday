# WITbragday
## Analysis of #WITbragday tweets

This folder contains a csv of all the tweets containing `#WITbragday`. It can be subset to include only unique tweets by filtering for `is_retweet == FALSE`. Also included in this folder is the markdown with all of my analysis work and a markdown/slidy document with only the work I presented at eUSR on November 3, 2017.
